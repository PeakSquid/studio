# **App Name**: IronRank

## Core Features:

- Lift and PR Tracking: Users can log personal records (PRs) for various lifts, including custom exercises. This feature enables viewing of historical progress and rankings, with data stored locally for quick access and offline use.
- AI Workout Plan Generator: Leverages an AI coach tool to formulate personalized, multi-week training programs (e.g., 12-week cycles). The AI tool uses reasoning based on user goals, current PRs, and historical data to intelligently tailor workout plans.
- AI Chat Coach: An interactive AI tool provides real-time guidance and answers to training questions. The coach analyzes user stats, lifts, ranks, and workout history to offer personalized advice and insights.
- Guided Workout Execution: Guides users through their scheduled workouts with clear exercise instructions, interactive set tracking (weight, reps, and completion), and an integrated rest timer with visual cues and adjustable intervals.
- Achievement & Progression System: Tracks user progress against a comprehensive, tier-based ranking system for individual lifts and overall performance. Users unlock achievements for reaching milestones, motivating continuous improvement.
- Workout Streak & Activity Dashboard: Displays a daily workout streak, overall workout count, and a visual heatmap of recent activity to encourage consistent engagement and provide a quick overview of training frequency.
- Personalized User Settings: Allows users to customize their profile including name, preferred weight units (lb/kg), and bodyweight, affecting calculations for certain lifts and rank tiers.

## Style Guidelines:

- Primary UI elements, progress indicators, and key interactive components feature a vibrant, high-energy yellow-green (#E8FF3A), symbolizing progress and vitality.
- The main background utilizes a deep, stark black (#0A0A0A) to establish a high-contrast dark theme, emphasizing the luminous primary color and making content pop.
- A secondary accent color, a warm orange (#FF6B35), is used for personal record (PR) indications and critical alerts, providing a distinct visual hierarchy.
- Headlines and prominent numeric displays employ 'Bebas Neue' (sans-serif), conveying a bold, athletic, and impactful presence.
- All body text and general UI elements use 'DM Sans' (sans-serif), chosen for its modern, clean lines, and excellent readability across various screen sizes.
- Minimalist SVG line icons are used for universal UI controls and navigation. Expressive emojis and symbolic glyphs augment these for achievements, muscle group representations, and status indicators, enhancing intuitive understanding.
- A mobile-first, card-based interface structure is utilized, featuring distinct, rounded content blocks for clear information partitioning. A persistent bottom tab bar facilitates primary navigation, complemented by sliding modals for focused interactions and dynamic content presentation via flexible grid/flexbox systems.
- Subtle yet impactful animations and transitions are implemented using custom cubic-bezier timing functions, providing smooth feedback for modal interactions, button presses, and achievement unlocks. Dynamic progress bar fills and celebratory badge 'pop' animations further enrich the user experience during workout completions and milestone achievements.
